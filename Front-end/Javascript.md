# Javascript学习

## 简介

Js最初本身就是一个针对网页的开发语言，并且开发时间很短。所以这个开发语言很烂。非常有用，但是并不美观。并且内容也非常分散不好描述。

## 语法

### 数据类型

Javascript本身就是一门有用动态类型的的语言，同意变量可以用作不同类型。但Javascript本身也是存在基本数据类型的。

**基本类型**

- String
- Number
- Boolean
- Null
- Undefined
- Symbol

**对象类型**

- Object
- Array
- Function
- 以及特殊对象RegExp（正则）和Date（日期）

### 注释

\/\/代表单行注释

\/\* \*\/代表多行注释

```javascript
//这是单行注释

/*
这是多行注释
*/
```
